# AI Infrastructure HTML Portfolio

A curated HTML portfolio of AI infrastructure research visuals, built for interview demos and public-facing sharing.

The portfolio brings together work across:

- semiconductor systems and supply constraints
- power, cooling, and grid bottlenecks
- AI infrastructure landscape maps
- revenue flow and market visualizations

## Live Site

GitHub Pages:

`https://pickle1024.github.io/ai-infrastructure-html-portfolio/`

## What's Inside

- `docs/index.html`
  GitHub Pages entry point
- `docs/portfolio/ai-infrastructure-portfolio.html`
  Main portfolio viewer
- `docs/portfolio/infra系列/`
  Semiconductor and supply-side slides
- `docs/portfolio/portfolio-assets/constraints-html/`
  Infrastructure constraints slides
- `docs/portfolio/sankey chart/sankey chart html/`
  Revenue flow visualizations
- `docs/portfolio/ai-infra-logos/`
  Local logo assets used by the landscape page

## Notes

- The site is intentionally curated rather than exhaustive.
- Only presentation-friendly horizontal pages are included.
- Some slides use external logo or CDN resources, so internet access improves rendering fidelity.
