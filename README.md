# AI Infrastructure Research Portfolio

This is a self-contained static site for showcasing selected AI infrastructure research visuals and HTML-based portfolio pieces.

## Structure

- `docs/index.html`
  GitHub Pages entry point
- `docs/portfolio/ai-infrastructure-portfolio.html`
  Main portfolio viewer
- `docs/portfolio/infra系列/`
  Core semiconductor overview slides
- `docs/portfolio/portfolio-assets/constraints-html/`
  AI infrastructure constraints slides
- `docs/portfolio/sankey chart/sankey chart html/`
  Revenue flow visualizations
- `docs/portfolio/ai-infra-logos/`
  Local logo assets used by the landscape page

## Deploy To GitHub Pages

1. Create a GitHub repository.
2. Upload this project as-is.
3. In GitHub, open `Settings` -> `Pages`.
4. Under `Build and deployment`, choose `Deploy from a branch`.
5. Select branch `main` and folder `/docs`.
6. Save.

Your site will then be available at:

`https://<your-github-username>.github.io/<repo-name>/`

## Notes

- The site is curated for interview demos.
- Only horizontal pages were included to keep the presentation cleaner.
- Some pages use remote logo services or CDN scripts, so internet access is still needed for those external resources.
